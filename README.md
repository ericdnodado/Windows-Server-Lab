Windows Server Lab (Active Directory + Enterprise Services)

📌 Overview

This project is a hands-on Windows Server lab designed to simulate a real-world enterprise environment. It demonstrates the deployment, configuration, and management of core infrastructure services such as:

- Active Directory Domain Services (AD DS)
- DNS & DHCP
- Group Policy (GPO)
- File Services (NTFS & Shared Folders)
- IIS Web Server
- Basic Security Hardening & Auditing

This lab is built to showcase system administration, troubleshooting, and infrastructure design skills.

🎯 Objectives

- Build a fully functional Active Directory domain
- Configure centralized authentication and authorization
- Implement Group Policy for security and user control
- Deploy network services (DNS, DHCP)
- Simulate real IT support scenarios
- Practice PowerShell automation

🏗️ Lab Architecture

🔹 Topology
2x Domain Controller (DC01, DCO2)
1x File Server (FS01)
1x Web Server (WEB01)
1x Windows Client (CL01)
